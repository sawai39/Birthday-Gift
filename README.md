# 生日惊喜网页

一个为生日准备的惊喜网页，包含刮刮乐效果和礼物详情展示。

## 功能特点

1. **刮刮乐效果**：卡片初始被马赛克遮挡，需要用手指滑动刮开
2. **卡通风格icon**：每个卡片展示对应的卡通风格icon
3. **礼物详情**：刮开后可点击进入详情页，展示真实礼物实拍图
4. **响应式设计**：适配iPhone17手机屏幕
5. **温馨配色**：使用粉色系渐变背景，营造浪漫氛围

## 礼物列表

- 🎂 生日蛋糕
- 🧘‍♀️ Lululemon瑜伽裤
- 😷 防晒口罩
- 👠 高跟鞋
- 🎥 生日视频

## 技术实现

- **HTML5**：页面结构
- **CSS3**：样式设计，包括响应式布局
- **JavaScript**：交互功能，包括刮刮乐效果和详情页展示

## 如何运行

1. 克隆项目到本地
2. 用浏览器打开 `index.html` 文件即可
